# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-02

### Added
- Initial release
- Multi-account connection and management
- Tradovate API integration
- Authentication and token management
- Account balance and position tracking
- Order placement and management
- Multi-account operations (parallel execution)
- Portfolio management tools
- Risk analysis and metrics
- Utility functions for common tasks
- WebSocket client (template)
- Advanced examples and use cases
- Comprehensive documentation in Hebrew
- Auto token renewal
- Error handling and retry logic
- Dashboard and status reporting
- Export functionality for portfolio data
- Monitoring mode with configurable intervals

### Features
- **TradovateClient**: Single account API client
- **MultiAccountManager**: Manage multiple accounts simultaneously
- **Utils**: Helper functions for formatting, calculations, risk management
- **Portfolio Manager**: Advanced portfolio analysis and monitoring
- **Examples**: Basic and advanced usage examples

### Supported Operations
- Connect to multiple Tradovate accounts
- Get account information and balances
- Fetch positions and orders
- Place and cancel orders
- Execute operations on all accounts at once
- Risk analysis across accounts
- Real-time monitoring
- Export portfolio snapshots

### Documentation
- README.md (English/Hebrew)
- GUIDE.md (Detailed Hebrew guide)
- Inline code documentation
- Configuration examples

### Security
- Config file not tracked in git
- Environment variable support
- Secure credential handling
