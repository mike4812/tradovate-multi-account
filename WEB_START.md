# 🌐 הפעלת האתר - מדריך מהיר

## 🚀 התחלה מהירה

### 1️⃣ התקן חבילות (פעם אחת)
```powershell
npm install
```

### 2️⃣ הפעל את השרת
```powershell
npm run web
```

### 3️⃣ פתח את הדפדפן
```
http://localhost:3000
```

**זהו! האתר רץ! 🎉**

---

## 📸 מה תראה באתר?

### 🏠 דף הבית
- Dashboard עם סיכום כל החשבונות

- כרטיס לכל חשבון
- יתרה ופרטים

### 📈 פוזיציות
- טבלה של כל הפוזיציות הפתוחות
- רווח/הפסד בזמן אמת
- מידע מפורט

### 📋 הזמנות
- רשימת הזמנות פעילות
- סטטוס כל הזמנה

### ⚙️ הגדרות
- רענון אוטומטי
- מעבר בין Demo ל-Live
- התנתקות

## 🔧 פקודות שימושיות

```powershell
# הפעלת האתר
npm run web

# או
npm run dev

# הפעלה עם פורט מותאם
$env:PORT=8080; npm run web

# עצירת השרת
Ctrl+C
```

## 📱 גישה ממכשירים אחרים

אם אתה רוצה לגשת לאתר ממכשיר אחר ברשת המקומית:

1. **מצא את ה-IP שלך:**
```powershell
ipconfig
# חפש את IPv4 Address
```

2. **פתח בדפדפן של המכשיר האחר:**
```
http://YOUR_IP:3000
# לדוגמה: http://192.168.1.100:3000
```

## ❓ שאלות נפוצות

### האתר לא נפתח?
1. וודא שהשרת רץ (תראה הודעה ב-terminal)
2. בדוק שהפורט 3000 לא תפוס
3. נסה לפתוח ב-incognito mode

### הכפתורים לא עובדים?
פתח Developer Tools (F12) ובדוק את ה-Console לשגיאות.

### רוצה לשנות את העיצוב?
ערוך את `web/css/style.css`

### רוצה להוסיף פיצ'רים?
ערוך את `web/js/app.js`
